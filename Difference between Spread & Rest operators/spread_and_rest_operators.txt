ðŸ”¹ Rest Operator (...)

	â€¢	The rest operator collects multiple values into an array.
	â€¢	It is mainly used in function parameters to gather the remaining arguments into an array.



ðŸ“Œ Example 1: Collecting Function Arguments

- Here, a = 10, b = 20, and the rest of the arguments [30, 40, 50] are stored in the rest array.
